# スクリーンショット撮影指示（ProtoPedia 画像・最大5枚）

ProtoPedia は最大5枚登録でき、**1枚目がメイン画像**（SNS/一覧のサムネイル）になります。
推奨サイズ **880×495px**（16:9）、png または jpg。撮影は本番アプリ `https://app.eventtune.link` またはローカル起動で。

## 撮影の共通ルール
- ブラウザ幅を 1280px 以上にしてから撮り、**880×495（16:9）にトリミング**。
- サンプルデータ（`sample_data/`）を投入したデモ用スペースで撮る。**実顧客の個人情報は写さない**。
- 文字が小さくなりすぎないよう、必要なUI要素だけが写る構図に寄せる。
- ダーク/ライトはプロダクトの標準テーマで統一。

## 審査軸との対応
「ユーザビリティ」「実用性・体験価値」の直接の証拠になる。**画面に"AIの判断根拠"が写っているカット（4・5枚目）を必ず1枚は入れる**。

---

## 1枚目（メイン / 必須）— ヒーロー
- **狙い**：一目で「イベントデータ → 根拠つきフォロー＆分析」と伝える顔。
- **候補A（推奨）**：`architecture.png`（作成済みの構成図）をそのままメインにする。情報量が多く、審査員に全体像が即伝わる。
- **候補B**：`/agent` のチャット画面 + サイドに統合済みデータ、上部にタグライン「出会いを整え、成果へチューニングする。」をオーバーレイした合成画像。
- どちらか1つ。迷ったら **A（構成図）**。

## 2枚目 — ファイル投入（つくる/使いやすさ）
- **画面**：`/agent`（チャット）画面に **Excel/CSV・Word・PDF・PowerPoint** を**ドラッグ&ドロップしている瞬間**（ドロップゾーンがハイライトされた状態）。
- **写すもの**：複数ファイル（名簿・アンケート・費用など）を同時に投げ込んでいる様子、または取り込みプランの提案が出て**確認（承認）ボタンが並ぶ**ところ（「承認したプランがそのまま実行される」ことが伝わる構図）。
- **キャプション例**：「バラバラのファイルを、そのまま投げ込むだけ」

## 3枚目 — 意味統合の結果（オントロジー）
- **画面**：`/data`（探索可能な3ペインのデータ画面：集計・AIサマリ・列管理・並べ替え/フィルタ）で、統合後の Person・Event・Product などが表示されている画面。
- **写すもの**：列名がバラバラだった入力が、5マスタ＋ファクトに正規化され、**FK が関連マスタ名で結合表示**されている様子。可能なら `appeal_summary`（自然文要約）や AI サマリが見えるレコード詳細。
- **キャプション例**：「列がバラバラでも、AIが"意味"で統合」

## 4枚目 — チャット分析 + Code Interpreter（実用性・監査可能性）★根拠カット
- **画面**：MarketingAgent に「先月比で費用対効果を出して」と聞き、**AIが生成したPythonコードと実行結果（表/数値）が両方表示**されているところ。
- **写すもの**：コードブロック＋結果。実データに基づく回答であることが分かる構図。
- **キャプション例**：「"それっぽい数字"ではなく、実データ上でPythonを実行して回答」

## 5枚目 — 個別フォロー生成 + 根拠パネル（体験価値）★根拠カット
- **画面**：個別方式（少人数＝1人ずつ実名フル生成）またはセグメント方式（多数＝軸で分類）で生成されたフォロー文面と、各ブロックの **`reason_for_inclusion`（なぜこの人にこの内容か）** が見えるパネル。可能なら方法・到達可否を確認するゲートや、**チャットで「ok」と返して承認する HIL**（ボタン＝チャット送信と同義）も。
- **写すもの**：1人分のフォロー文面＋採用理由（根拠は個別・セグメントの両方式とも必須）。「100人に100通り」を象徴するリスト表示でも可。
- **キャプション例**：「どの一文にも"なぜこの人にこの内容か"が残る（承認してから配信）」

## トリミング補助（任意）
1280×720 で撮った画像を 880×495 にするには:
```
sips -z 495 880 raw.png --out shot.png     # 縦横比が16:9なら単純リサイズでOK
# 16:9でない場合は先に16:9へクロップしてからリサイズ
```

## 撮影後チェック
- [ ] 5枚とも 880×495 前後・16:9
- [ ] 実顧客の個人情報が写っていない（サンプルデータのみ）
- [ ] 4枚目 or 5枚目に「AIの判断根拠」が写っている
- [ ] 1枚目（メイン）だけで作品の主旨が伝わる
